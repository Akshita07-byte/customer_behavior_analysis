# 📊 Data Analytics End-to-End Project

## 📌 Overview

This project demonstrates a complete **Data Analytics workflow**, starting from raw data handling to generating insights through SQL, Python, and visualization tools. It includes data cleaning, exploratory data analysis (EDA), database operations, dashboard creation, and final reporting.

The goal is to analyze customer behavior data and extract meaningful business insights to support decision-making.

---

## 📂 Dataset

* Customer Shopping Behavior Dataset
* Contains information such as:

  * Customer ID
  * Age, Gender
  * Purchase Amount
  * Category, Item Purchased
  * Discount Applied
  * Payment Method
  * Subscription Status
  * Shipping Type

---

## 🛠️ Tools & Technologies Used

* **Python (Pandas, NumPy)** – Data cleaning & EDA
* **Jupyter Notebook** – Analysis environment
* **SQL (PostgreSQL / MySQL / SQL Server)** – Data querying & analysis
* **Power BI** – Dashboard creation & visualization
* **Gamma AI** – Presentation (PPT) creation
* **Excel (optional)** – Data inspection
